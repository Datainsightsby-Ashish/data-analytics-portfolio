# Data Analytics Portfolio

This repository contains end-to-end data analytics projects focused on EDA, business insights, and advanced analysis.

## Projects
1. **Exploratory Data Analysis** — Netflix titles: cleaning, trends, and visual insights  
   Folder: `01_EDA_Project`
2. **Business Insights** — Superstore sales: KPIs, profitability, and recommendations  
   Folder: `02_Business_Insights`
3. **Advanced Analysis Project** — To be decided (IPL analysis or customer segmentation)  
   Folder: `03_Final_Project`

## Tech
Python · Pandas · NumPy · Matplotlib · Seaborn · Jupyter/Colab

## How to Run
- Open the `.ipynb` files in Google Colab or Jupyter.
- Data files are ignored in Git (`/data`), with links or instructions provided in each project folder.
